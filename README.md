## 🎯 Overview

A complete end-to-end **Machine Learning system** that classifies news articles as **Real** or **Fake** using Natural Language Processing (NLP). Built as a Final Year project, it covers the full ML pipeline — from raw data to a deployed interactive web application.

---

## ✨ Features

| Feature | Details |
|---|---|
| **Dataset** | Kaggle Fake & Real News (≥ 40 000 articles) |
| **NLP Pipeline** | Lowercase → URL removal → Stopword removal → Stemming |
| **Feature Extraction** | TF-IDF Vectorizer (unigrams + bigrams) |
| **Models** | Logistic Regression, Naive Bayes, Random Forest |
| **Best Accuracy** | ~94–95% |
| **Cross Validation** | 5-Fold Stratified K-Fold |
| **Web App** | Streamlit with dark/light mode, sidebar navigation |
| **Confidence Score** | Probability-based prediction confidence |

## 🚀 How to Run

### Step 1 – Clone & Setup Environment

```bash
# Clone the project
git clone <your-repo-url>
cd fake_news_project

# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows

# Install dependencies
pip install -r requirements.txt
```

### Step 2 – Get the Dataset

**Option A (Recommended):** Download from Kaggle  
👉 https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

Place `Fake.csv` and `True.csv` inside the `notebook/` folder.

**Option B:** The notebook auto-generates a 4 000-row synthetic dataset if no CSV is found (good for testing, lower accuracy).

### Step 3 – Train the Model

```bash
cd notebook
jupyter notebook fake_news_model.ipynb
```

Run all cells top to bottom. This will:
- Load & preprocess the dataset
- Generate EDA charts (saved as PNG files)
- Train 3 ML models
- Evaluate and compare them
- Save `model.pkl`, `vectorizer.pkl`, and `model_metadata.pkl` to the project root

### Step 4 – Launch the Streamlit App

```bash
# From the project root
cd ..
streamlit run app.py
```

Open your browser at **http://localhost:8501**

## 🧠 NLP Pipeline Explained

```
Raw Text
   ↓
Lowercase                     "SHOCKING news..." → "shocking news..."
   ↓
Remove URLs                   "visit https://..." → "visit "
   ↓
Remove HTML & Special chars   "<b>text</b>" → "text"
   ↓
Tokenization                  "fake news" → ["fake", "news"]
   ↓
Stopword Removal              removes: "the", "is", "in", "a", ...
   ↓
Porter Stemming               "running" → "run", "government" → "govern"
   ↓
TF-IDF Vectorization          text → numerical feature vector
   ↓
ML Classifier                 feature vector → Real / Fake
```
